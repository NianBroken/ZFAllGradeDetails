# 正方教务系统导出成绩详情

<img src="https://raw.githubusercontent.com/NianBroken/ZFAllGradeDetails/main/img/1.png" style="zoom:60%;" />

## 简介

ZFAllGradeDetails 可以绕过正方教务管理系统默认权限限制，一键下载包含平时成绩、期末成绩及最终成绩的完整成绩单。

## 先决条件

1. 首先在浏览器中安装 [Tampermonkey](https://www.tampermonkey.net/) 插件。
2. 接着在 Tampermonkey 中安装 [ZFAllGradeDetails]https://greasyfork.org/scripts/542632) 脚本。

## 使用方法

1. 使用安装了Tampermonkey和ZFAllGradeDetails的浏览器进入你的教务系统。
2. 进入“学生成绩查询”页面。
3. 点击浏览器工具栏中的Tampermonkey图标。
4. 点击“导出成绩详情”。
5. 打开下载后的表格文件。
